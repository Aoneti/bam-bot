# Birds Around Me

**Бот для отслеживания птиц рядом с вами https://t.me/birdsaroundmebot**  
Работает на данных **iNaturalist.org**. Показывает, кто летает вокруг, уведомляет о редких видах и птицах из вишлиста.

## Основные возможности

- **🐦 Сканировать** — список видов в радиусе
- **🗒 Карточка птицы** — фото, расстояние, время наблюдения, редкость, семейство и отряд
- **⭐ Вишлист** — добавляйте птиц. Бот пришлёт уведомление, как только они появятся рядом
- **🚨 Проверить редких птиц** — мгновенная проверка уникальных наблюдений
- **🔥 Bird Alert** — показывает птицу, появившуюся в округе впервые за 90+ дней
- **🔍 Inline-режим** — `@birdsaroundmebot скворец` в любом чате
