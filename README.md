# Birds Around Me

**Бот для отслеживания птиц рядом с вами → https://t.me/birdsaroundmebot**  
Работает на данных **iNaturalist** и **eBird (Cornell Lab)**. Показывает, кто летает вокруг, определяет птицу по фото, уведомляет о редких видах и птицах из вишлиста.

## Основные возможности

- **🐦 Сканировать** — список видов в радиусе с числом встреч
- **📷 Определить птицу** — пришлите фото, бот определит вид через iNaturalist Computer Vision (топ-5 вариантов с уверенностью в %)
- **🗒 Карточка птицы** — фото, расстояние, время наблюдения, редкость, семейство, отряд, сезонность, флаги инвазивности/эндемизма
- **⭐ Вишлист** — добавляйте птиц. Бот пришлёт уведомление, как только они появятся рядом
- **🚨 Редкие птицы** — мгновенная проверка: охраняемые виды (CR/EN/VU) из iNaturalist + необычные наблюдения, проверенные ревьюерами eBird
- **🔥 Bird Alert** — уведомляет, если вид появился в округе впервые за 90+ дней
- **🌿 Сезонный фильтр** — показывает только птиц, характерных для текущего сезона
- **🔍 Inline-режим** — `@birdsaroundmebot скворец` в любом чате, с данными о ближайшем наблюдении

## Технические детали

- **Архитектура**: единый файл, async (aiogram 3.x + aiohttp)
- **БД**: SQLite с WAL-режимом (aiosqlite)
- **Кэширование**: LRU-кэши для наблюдений, таксонов, гистограмм, геокодирования, хотспотов; общий кэш по координатам (пользователи из одного района делают 1 запрос вместо N)
- **Rate limiting**: семафор на API-запросы к iNaturalist, экспоненциальный backoff
- **Планировщик**: APScheduler, батчевая обработка пользователей
- **eBird**: опциональная интеграция (нужен API-ключ)
