# Birds Around Me

**Telegram-бот для отслеживания птиц рядом с вами**  
Работает на данных **iNaturalist.org**. Показывает, кто летает вокруг, уведомляет о редких видах и птицах из вишлиста.

[![Python](https://img.shields.io/badge/Python-3.11%2B-blue.svg)](https://www.python.org)
[![aiogram](https://img.shields.io/badge/aiogram-3.x-orange.svg)](https://docs.aiogram.dev)
[![iNaturalist](https://img.shields.io/badge/API-iNaturalist-green.svg)](https://api.inaturalist.org)

---

## Основные возможности

- **🐦 Сканировать** — список видов в радиусе
- **🗒 Карточка птицы** — фото, расстояние, время наблюдения, редкость, семейство и отряд
- **⭐ Вишлист** — добавляй птиц, бот пришлёт уведомление, как только они появятся рядом
- **🚨 Проверить редких птиц** — мгновенная проверка уникальных наблюдений
- **🔥 Bird Alert** — показывает птицу, появишуюся в округе впервые за 90+ дней
- **📩 Автоматические проверки** — каждый час для всех пользователей
- **🔍 Inline-режим** — `@birdsaroundmebot скворец` в любом чате
